# Basic Chatbot (Python)

A simple rule-based chatbot built using Python.
The chatbot responds to predefined user inputs and demonstrates the use of conditional statements, loops, and functions.

This project was developed as part of the **CodeAlpha Python Programming Internship**.

---

## Features

* Simple console-based chatbot
* Responds to predefined messages
* Uses `if-elif` conditions
* Runs continuously until the user types **bye**
* Beginner-friendly Python implementation

---

## Technologies Used

* Python
* Functions
* Loops
* Conditional Statements
* Console Input / Output

---

## How to Run

1. Navigate to the project folder.

2. Run the chatbot:

```bash
python basic_chatbot.py
```

---

## Example Interaction

```
Simple Chatbot
Type 'bye' to exit the chat.

You: hello
Bot: Hi! How can I help you?

You: how are you
Bot: I'm fine, thanks! How about you?

You: bye
Bot: Goodbye! Have a great day.
```
